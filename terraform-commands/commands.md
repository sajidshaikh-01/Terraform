1. Initialize Terraform
terraform init
---
2. Validate Terraform Code
terraform validate
---
3. Format Code Properly
terraform fmt
---
4. See What Will Change (Dry Run)
terraform plan
---
5. Apply Changes
terraform apply
---
6. Destroy Infrastructure
terraform destroy
---
7. Output Values
terraform output
---
8. Manage Terraform State
terraform state list
---
9. Terraform Workspaces (Environments)
terraform workspace new dev
---
10. Show State & Resources
terraform show
---
11. Dependency Graph
terraform graph
---
12. Provider Information
terraform providers
---
13.refresh the script
terraform apply -refresh-only
